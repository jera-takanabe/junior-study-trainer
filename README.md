# Junior Study Trainer

Junior Study Trainer は、中学生向けのタイマー付き一問一答・反復学習アプリです。

学校教科、漢検、英検、数検、その他の拡張教材を、同じ形式で扱えることを目指します。

現在は、プロトタイプ教材として漢検5級を実装しています。

## 現在の位置づけ

- アプリ本体は汎用学習トレーナーとして設計する
- 漢検5級は最初のプロトタイプ教材として扱う
- 将来的に、学校教科・英検・数検・その他教材を追加できる構成にする

## 現在の対象

- 漢検5級
- Set 001〜010
- 合計438問予定

## 今後の方針

- まずはローカルHTML + JavaScriptで運用する
- 問題データは setId / category / level / targets を持つ形式で管理する
- 途中の気づきや後回し事項は TODO.md に記録する
- 将来的にWebサーバー配置やAWS配置を検討する
