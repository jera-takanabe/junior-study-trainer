# 英語 第1回定期テスト 問題設計

## 基本方針

教科書ページを基準にし、めきめきEnglish、めきめき英単語、単元テストを参照して、定期テストで問われやすい形にする。

ただし、著作権配慮のため、教科書本文や単元テスト本文のまとまった引用は避ける。
問題は、語句、文法、語順、短い自作英文、応答表現を中心に構成する。

## セット分割

- e001: Get Ready
- e002: PROGRAM 1 語句
- e003: PROGRAM 1 be動詞
- e004: PROGRAM 1 本文理解・表現確認
- e005: PROGRAM 2 語句
- e006: PROGRAM 2 一般動詞
- e007: PROGRAM 2 本文理解・表現確認
- e008: PROGRAM 3 p.37-p.39 can / can't 基本

## 出題形式

- 日本語から英語
- 英語から日本語
- 空所補充
- 語順整序
- Yes/No疑問文への答え方
- 疑問詞 Where / When の使い分け
- am / are / do / don't / can / can't の選択
- 単語の意味確認
- 複数形の確認

## 難度

- 基本: 定期テストで必ず取りたい語句・文法
- 標準: 文の形、語順、応答まで確認する問題
- 発展: 本文理解・英作文に近い確認問題

## 注意点

PROGRAM 3 は教科書 p.39 までに含まれるが、単元テストの指定は PROGRAM 1 & 2 である。
そのため、PROGRAM 3 は can / can't の基本確認に限定し、問題数と難度を抑える。
