# 🏉 英検3級・動詞の活用マスターシート（規則動詞・不規則動詞）

## 🦖 1. 規則動詞の4つのルール
基本はうしろに **-ed** をつけるだけ！形が変わる4つの例外（ラグビールール）を覚えよう。

| ルール | 内容 | 具体例 |
| :--- | :--- | :--- |
| **① 基本形** | そのまま **ed** をつける | play → play**ed**<br>start → start**ed**<br>watch → watch**ed** |
| **② -e で終わる** | **d** だけをつける | like → like**d**<br>use → use**ed**<br>live → live**d** |
| **③ 子音 ＋ y** | **y を i に変えて ed** | study → stud**ied**<br>try → tr**ied**<br>cry → cr**ied**<br>※playはyの前が「a(母音)」なのでplayedでOK |
| **④ 短母音＋子音** | **最後の子音を重ねて ed** | stop → stop**ped**<br>plan → plan**ned**<br>drop → drop**ped** |

💡 **耳で覚える発音ルール（ヤマハの貯金を使おう！）**
* 「ド」の音：play**ed**（プレイド） / open**ed**（オープンド）
* 「ト」の音：lik**ed**（ライクト） / wash**ed**（ウォッシュト）
* 「イド」の音：start**ed**（スターティッド） / visit**ed**（ヴィジティッド）

## 🏃 2. 不規則動詞の活用（英検3級・重要50選）
形が変わる不規則動詞は、変化のパターン（リズム）で覚えるのが最大効率です。
※過去形と過去分詞形が同じものは、ライティングで使いやすいため超重要です。
