## A-8. targets 方針

targets には、以下を入れる。

```text
1. 対象漢字1字
2. 熟語・語句
3. 同音異字や誤字訂正で比較対象になる語
```

例：

```javascript
targets: ["価", "値", "価値"]
```

同音異字の場合：

```javascript
targets: ["準", "備", "準備", "順備"]
```

注意：

```text
・1026字の網羅は漢字1字の targets で確認する
・苦手語句は熟語 targets で確認する
・誤字や混同語も targets に入れてよい
```

正式IDは以下で統一する。


例：


注意：

```text

```

## B-4. 対象漢字

Set 002 の対象漢字は以下の20字とする。

```text
価・値・態・度・姿・勢・規・則・制・度
判・断・可・能・原・因・改・善・備・準
```

### 補足

`度` は `態度` と `制度` の両方に登場するため、対象漢字リスト上は重複している。

教材上は以下のように扱う。

```text
態度：態・度
制度：制・度
```

ただし、1026字の網羅管理では `度` は1字として扱う。

Set 002 の実質的なユニーク対象漢字は以下である。

```text
価・値・態・度・姿・勢・規・則・制
判・断・可・能・原・因・改・善・備・準
```

つまり、ユニーク漢字としては19字である。

今後、Set 003以降で不足分を調整する。

## B-8. targets 方針


## B-12. 注意点・見直し候補

### 1. 対象漢字数について

Set 002 は、定義上20字としていたが、`度` が重複しているため、ユニーク漢字としては19字である。

これは致命的な問題ではないが、今後の網羅管理では注意する。

対応案：

```text
案A：Set 002はこのまま運用し、Set 003以降で1字分を調整する
案B：Set 002の対象漢字を1字追加して20字にする
```

現時点では、問題データ作成済みのため、案Aを採用する。

### 2. 補助語彙の扱い

`基準`、`不可能`、`結果`、`用意`、`決まり` などは、Set 002 の中核漢字以外を含む場合がある。

ただし、語句理解・対義語・類義語として必要なため、補助語彙として扱う。
