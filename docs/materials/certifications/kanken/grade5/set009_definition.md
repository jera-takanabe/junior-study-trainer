## 3. 目的

Set 009 の目的は以下である。

```text
・学校や社会で使う集団・役割に関する語彙を増やす
・係、委員、代表、参加など、学校生活で使いやすい語を扱う
・社会科や説明文で登場しやすい語に慣れる
・読み、意味、書き取りをバランスよく練習する
・チームや集団の中での役割を説明できる語彙を増やす
```

特に、以下のような語を使えるようにする。

```text
社会
集団
役割
参加
代表
委員
係員
活動
協力
公共
```

## 4. 対象漢字

Set 009 の対象漢字は以下の20字とする。

```text
社・会・集・団・役・割・参・加・代・表
委・員・係・活・動・協・力・公・共・民
```

### 補足

`加` は Set 006 の `増加` でも扱っている。  
Set 009では `参加` として再登場させる。

`動` は Set 004 の `行動`、Set 008 の `移動` でも扱っている。  
Set 009では `活動` として再登場させる。

`力` は Set 004 の `努力` でも扱っている。  
Set 009では `協力` として再登場させる。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

## 8. targets 方針

Set 009 では、以下の両方を targets に入れる。

```text
・漢字1字
・熟語・語句
```

例：

```javascript
targets: ["社", "会", "社会"]
```

同音異字・誤字訂正では、誤字側や比較対象も targets に入れる。

例：

```javascript
targets: ["参", "加", "参加", "産加"]
```

## 12. 注意点・見直し候補

### 1. 学校生活との接続

Set 009では、学校生活で実際に使う語を多く扱う。

```text
委員
係員
代表
参加
協力
活動
```

これらは、日記・作文・学級活動の文章でも使いやすい。

### 2. 社会科との接続

`社会`、`公共`、`国民`、`地域` などは、社会科の文章理解にもつながる。

Set 008の `地域` とも関連するため、復習効果がある。

### 3. `公共` の扱い

`公共` はやや抽象的だが、中学以降でも重要な語である。

まずは以下のように扱う。

```text
みんなに関係するもの
社会全体に関係するもの
```

難しすぎる説明は避ける。
