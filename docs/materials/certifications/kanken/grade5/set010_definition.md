## 3. 目的

Set 010 の目的は以下である。

```text
・自然や環境に関する基本語彙を増やす
・生活や安全に関する語を扱う
・理科・社会・保健の文章理解につながる語彙を増やす
・読み、意味、書き取りをバランスよく練習する
・基本熟語・日常語シリーズ Set 001〜010 の一区切りとする
```

## 4. 対象漢字

Set 010 の対象漢字は以下の20字とする。

```text
自・然・環・境・安・全・危・険・健・康
栄・養・衛・生・災・害・防・止・保・護
```

### 補足

`全` は今後の「全部」「安全」などで頻出するが、Set 010では `安全` として正式に扱う。

`生` は Set 001 の `誕生` でも扱っている。  
Set 010では `衛生` として再登場させる。

`険` は Set 001 の `険しい` でも扱っている。  
Set 010では `危険` として再登場させる。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

## 8. targets 方針

Set 010 では、以下の両方を targets に入れる。

```text
・漢字1字
・熟語・語句
```

例：

```javascript
targets: ["自", "然", "自然"]
```

同音異字・誤字訂正では、誤字側や比較対象も targets に入れる。

例：

```javascript
targets: ["危", "険", "危険", "危検"]
```

## 12. 注意点・見直し候補

### 1. 安全・危険の扱い

`安全` と `危険` は対義語として扱いやすい。

```text
安全 ⇔ 危険
```

ただし、問題文は怖がらせる内容にせず、以下のように扱う。

```text
安全を確認する
危険な場所に近づかない
```

### 2. 健康・栄養・衛生の扱い

`健康`、`栄養`、`衛生` は日常生活・保健・成長にも関係する語である。

```text
健康な体
栄養をとる
衛生に気をつける
```

今後の「体・健康・成長」系セットにもつながる。

### 3. 自然・環境・災害の扱い

`自然`、`環境`、`災害`、`防止`、`保護` は、理科・社会・防災教育にもつながる。

Set 010ではまず基本熟語として扱い、後続セットで詳しく広げる。
