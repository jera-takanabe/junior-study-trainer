## 3. 目的

Set 006 の目的は以下である。

```text
・数量や程度を表す語彙を増やす
・範囲、単位、基準など、学習全般で使う語を扱う
・読み、意味、書き取りをバランスよく練習する
・算数/数学/理科/社会の文章理解につながる語彙を増やす
・説明文やグラフ、表の読み取りで使いやすい語を増やす
```

特に、以下のような語を使えるようにする。

```text
数量
程度
範囲
単位
基準
平均
割合
増加
減少
以上
```

## 4. 対象漢字

Set 006 の対象漢字は以下の20字とする。

```text
数・量・程・度・範・囲・単・位・基・準
平・均・割・合・増・加・減・少・以・上
```

### 補足

`度` は Set 002 の `態度`・`制度` でも扱っている。  
Set 006では `程度` として再登場させる。

`単` は Set 001 の `簡単` でも扱っている。  
Set 006では `単位` として再登場させる。

`準` は Set 002 の `準備`、Set 005 の `基準` 関連でも扱っている。  
Set 006では `基準` として正式に扱う。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

## 12. 注意点・見直し候補

### 1. 他セットとの重複

Set 006では、既出漢字がいくつか再登場する。

```text
度：Set 002でも使用
単：Set 001でも使用
準：Set 002でも使用
```

これは、別熟語で使えるようにするために許容する。

### 2. 算数・数学・理科との接続

Set 006は、漢字学習でありながら、算数・数学・理科の文章読解にもつながる。

```text
平均
割合
増加
減少
以上
以下
範囲
単位
```

これらは、今後の数学・理科カテゴリをアプリに追加するときにも再利用しやすい語彙である。

### 3. `以上`・`以下`・`以内` の混同

`以上`、`以下`、`以内` は混同しやすい。

```text
以上：その数を含んで、それより上
以下：その数を含んで、それより下
以内：決められた範囲の内側
```

Set 006では、意味選択や誤字訂正で扱う。
