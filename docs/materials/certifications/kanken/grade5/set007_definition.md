## 4. 対象漢字

Set 007 の対象漢字は以下の20字とする。

```text
感・情・謝・反・省・賛・成・対・意・見
思・決・満・足・不・満・喜・怒・悲・楽
```

### 補足

`成` は Set 004 の `成功` でも扱っている。  
Set 007では `賛成` として再登場させる。

`対` は Set 004 の `対象` でも扱っている。  
Set 007では `反対` として再登場させる。

`意` は既出語の説明文には登場しているが、Set 007では `意見`・`意思` で正式に扱う。

`満` は `満足` と `不満` の両方に登場するため、同一セット内で重複する。教材上は以下のように扱う。

```text
満足：満・足
不満：不・満
```

ただし、1026字網羅管理では `満` は1字として扱う。

## 12. 注意点・見直し候補

### 1. 感情語の扱い

Set 007では、感情に関わる語を扱う。

```text
感情
喜ぶ
怒る
悲しい
楽しい
不満
満足
```

ただし、問題文では子どもを責める表現にしない。

望ましい例：

```text
自分の気持ちを言葉にする
試合の結果に満足する
失敗を反省して次に生かす
```

避けたい例：

```text
君は反省しなさい
不満ばかり言う
```

### 2. `意思` と `意志`

`意思` と `意志` は混同しやすい。

```text
意思：考えや思い
意志：何かをやりぬこうとする心
```

Set 007では、まず `意思` を中核語彙として扱う。  
`意志` は混同しやすい比較語として、必要に応じて同音異字問題で扱う。

### 3. `反省` と `反対`

`反` は `反省` と `反対` で使う。

```text
反省：自分の行動をふり返ること
反対：逆の立場をとること
```

同じ漢字でも意味の広がりがあることを確認する。
