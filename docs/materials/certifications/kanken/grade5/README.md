# 漢検5級 プロトタイプ教材

## 1. 位置づけ

このフォルダは、Junior Study Trainer における最初のプロトタイプ教材である「漢検5級」の教材設計ドキュメントを管理する。

本プロジェクトは漢検専用ではなく、学校教科・漢検・英検・数検・拡張教材を扱う汎用学習トレーナーを目指す。

漢検5級は、そのデータ形式・セット運用・進捗管理・復習設計を検証するための最初の教材である。

## 2. 現在の範囲

現在は、以下を対象とする。

- 漢検5級
- Set 001〜010
- 基本熟語・日常語シリーズ
- 合計438問予定

## 3. 管理するもの

このフォルダでは、主に以下を管理する。

- セット定義ドキュメント
- 50セット構成の考え方
- 教材設計方針
- 後から見直すべき注意点

## 4. 問題データとの関係

実際の問題データは、以下で管理する。

- data/certifications/kanken/grade5/sets/
- data/certifications/kanken/grade5/bundled/

この docs フォルダは、問題データそのものではなく、教材設計と判断根拠を管理する。
