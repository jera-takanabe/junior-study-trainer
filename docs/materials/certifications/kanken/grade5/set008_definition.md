## 3. 目的

Set 008 の目的は以下である。

```text
・場所や位置を表す語彙を増やす
・方向や移動に関する語を扱う
・地図、道案内、学校生活、社会科の文章理解につながる語を増やす
・読み、意味、書き取りをバランスよく練習する
```

## 4. 対象漢字

Set 008 の対象漢字は以下の20字とする。

```text
位・置・方・向・道・路・通・過・移・動
到・着・出・発・周・囲・中・央・地・域
```

### 補足

`方` は Set 003 の `方法` でも扱っている。  
Set 008では `方向` として再登場させる。

`動` は Set 004 の `行動` でも扱っている。  
Set 008では `移動` として再登場させる。

`囲` は Set 006 の `範囲` でも扱っている。  
Set 008では `周囲` として再登場させる。

`中` は Set 005 の `途中` でも扱っている。  
Set 008では `中央` として再登場させる。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

## 12. 注意点・見直し候補

### 1. 方向・位置語の混同

`位置`、`方向`、`中央`、`周囲` は似た場面で使うが、意味が異なる。

```text
位置：ものがある場所
方向：向き
中央：まんなか
周囲：まわり
```

意味選択や類義語問題で区別できるようにする。

### 2. 移動語のまとまり

Set 008では、移動に関する語をまとめて扱う。

```text
移動
出発
到着
通過
```

作文や日記でも使いやすい語である。

### 3. 社会科・地図との接続

`地域`、`道路`、`方向`、`中央` は、社会科や地図の読み取りにもつながる。

今後、理科・社会カテゴリを追加する場合にも再利用しやすい。
