## 3. 目的

Set 004 の目的は以下である。

```text
・日常生活や学校生活で使う基本熟語をさらに増やす
・変化、比較、状態、行動を説明する語彙に触れる
・読み、意味、書き取りをバランスよく練習する
・作文や説明文で使いやすい語を増やす
・Set 001〜003で重複した対象漢字を補う
```

特に、以下のような語を使えるようにする。

```text
変化
比較
対象
状態
行動
努力
成功
失敗
希望
予定
```

## 4. 対象漢字

Set 004 の対象漢字は以下の20字とする。

```text
変・化・比・較・対・象・状・態・行・動
努・力・成・功・失・敗・希・望・予・定
```

### 補足

Set 004 では、Set 002・Set 003で発生したような同一セット内の重複を避け、20字すべてをユニーク漢字として扱う。

ただし、`態` は Set 002 の `態度` でも扱っているため、1026字網羅管理では重複カウントしない。

Set 004では `状態` の一部として再登場させ、同じ漢字を別熟語で使えるようにする意図がある。

## 8. targets 方針

Set 004 では、以下の両方を targets に入れる。

```text
・漢字1字
・熟語・語句
```

例：

```javascript

```

同音異字・誤字訂正では、誤字側や比較対象も targets に入れる。

例：


## 12. 注意点・見直し候補

### 1. Set 002との重複

`態` は Set 002 の `態度` でも扱っている。

Set 004では `状態` として扱う。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

### 2. `象` の扱い

`対象` の `象` は、単独ではやや抽象度が高い。

Set 004では、まず `対象` という熟語で扱い、補助語彙として `象徴` を入れる。

### 3. `失敗` の扱い

`失敗` は子どもにとってネガティブな語だが、日常語として重要。

問題文では、責める表現ではなく、以下のように扱う。

```text
失敗の原因を考える
失敗から学ぶ
```
