## 3. 目的

Set 005 の目的は以下である。

```text
・時間、順序、計画に関する語彙を増やす
・学習や生活管理で使いやすい熟語を扱う
・読み、意味、書き取りをバランスよく練習する
・予定や準備と関連する語を広げる
・説明文や日記、作文で使いやすい語を増やす
```

## 4. 対象漢字

Set 005 の対象漢字は以下の20字とする。

```text
時・間・期・限・順・序・計・画・整・理
確・認・連・続・途・中・開・始・終・了
```

### 補足

`認` は Set 001 の `承認` でも扱っている。  
Set 005では `確認` として再登場させる。

これは以下の意図で許容する。

```text
同じ漢字を別の熟語で使えるようにする
```

ただし、1026字網羅管理では重複カウントしない。

また、`計` は Set 004 の問題文にも一部登場しているが、Set 005では対象漢字として `計画` で正式に扱う。

## 12. 注意点・見直し候補

### 1. Set 001との重複

`認` は Set 001 の `承認` で扱っている。  
Set 005では `確認` として扱う。

```text
承認：よいと認めること
確認：たしかめること
```

これは、同じ漢字を別の熟語で使えるようにするため、許容する。

### 2. `期間` と `期限` の違い

`期間` と `期限` は混同しやすい。

```text
期間：ある始まりから終わりまでの間
期限：それまでに終えると決められた時点
```

Set 005では、意味選択や誤字訂正で扱う。

### 3. `開始` と `終了`

`開始` と `終了` は対義語として扱いやすい。

```text
開始 ⇔ 終了
```

直前対策でも使いやすい語彙なので、読み・意味・書き取りに入れる。
