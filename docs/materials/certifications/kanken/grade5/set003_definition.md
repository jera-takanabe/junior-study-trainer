## 3. 目的

Set 003 の目的は以下である。

```text
・説明文や学校生活でよく使う基本熟語を増やす
・理由、方法、目的、条件などを説明する語彙に触れる
・読み、意味、書き取りをバランスよく練習する
・文章の中で使いやすい抽象語を増やす
```

## 4. 対象漢字

Set 003 の対象漢字は以下の20字とする。

```text
目・的・方・法・条・件・必・要・重・要
関・係・説・明・記・録・観・察・実・験
```

### 補足

`要` は `必要` と `重要` の両方に登場するため、対象漢字リスト上は重複している。

教材上は以下のように扱う。

```text
必要：必・要
重要：重・要
```

ただし、1026字の網羅管理では `要` は1字として扱う。

Set 003 の実質的なユニーク対象漢字は以下である。

```text
目・的・方・法・条・件・必・要・重
関・係・説・明・記・録・観・察・実・験
```

ユニーク漢字としては19字である。

Set 002でも1字重複があったため、今後のSet 004以降では、対象漢字の重複を減らすか、補充漢字を入れて調整する。

## 8. targets 方針

Set 003 では、以下の両方を targets に入れる。

```text

・熟語・語句
```

例：

```javascript

```

同音異字・誤字訂正では、誤字側や比較対象も targets に入れる。

例：

```javascript

```

## 12. 注意点・見直し候補

### 1. 対象漢字の重複について

Set 003 では `要` が `必要` と `重要` で重複する。

```text
必要：必・要
重要：重・要
```

Set 002でも `度` が重複していたため、Set 004以降では、重複を避けるか、対象漢字の補充を行う。

### 2. Set 001との重複

Set 001でも `重` は `貴重` で扱っている。
Set 003では `重要` として再登場する。

これは「同じ漢字を別の熟語で使えるようにする」という意味では許容する。
ただし、1026字網羅管理では重複カウントしない。

### 3. 理科寄り語彙との関係

`観察`、`実験` は、将来の「自然・理科・数量語彙シリーズ」にも関係する。
ただし、中学準備語彙として重要なため、Set 003に前倒しで入れる。
