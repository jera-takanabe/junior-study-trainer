# 漢検5級 50セット構成ロードマップ v0.1

## 1. 目的

この文書は、漢検5級の対象漢字1026字を、タイムアタック一問一答アプリ用の教材セットとして段階的に整備するための、50セット前後の構成方針を定義する。

個別の `Set 001`、`Set 002` を場当たり的に作ると、後半で分類やタイトルが行き詰まる可能性がある。
そのため、最初に全体の分類軸を決め、各セットの役割を整理しておく。

---

## 2. 基本方針

### 2.1 Set は「テーマ別」だけでなく「シリーズ別」にする

50セットすべてに個別のユニークなテーマ名を付けると、後半で無理が出る。

そのため、以下のようにシリーズ化する。

```text
基本熟語・日常語 01
基本熟語・日常語 02
基本熟語・日常語 03
...
同音異字・使い分け 01
同音異字・使い分け 02
...
送り仮名・動詞 01
送り仮名・動詞 02
...
```

これにより、50セット規模でも命名が破綻しにくくなる。

---

## 3. Set の基本単位

1セットの目安：

```text
対象漢字：20字前後
問題数：40〜60問程度
```

ただし、同音異字・四字熟語・部首などは、必ずしも20字単位にならない場合がある。

## 7. この構成の意図

### 7.1 タイトル不足を避ける

50セットすべてに独自タイトルを付けるのではなく、シリーズ名 + 連番にする。

```text
基本熟語・日常語 01
基本熟語・日常語 02
...
```

これにより、後半でタイトル作りに困らない。

### 7.2 学習順を作れる

最初は日常語・学校語彙から入り、後半で同音異字・送り仮名・総合演習へ進む。

```text
基本語彙
↓
学校語彙
↓
抽象語
↓
理科・数量語彙
↓
使い分け
↓
送り仮名
↓
総合演習
```

### 7.3 1026字網羅と試験対策を両立する

前半〜中盤は、対象漢字を広くカバーする。
後半は、漢検形式に寄せた演習を増やす。

---

## 8. 1026字網羅との関係

このロードマップでは、1026字を次の2つの観点で管理する。

```text
1. targets に登場した漢字
2. 問題として出題された語句・熟語
```

対象漢字1026字の網羅は、`targets` によって確認する。

例：

```javascript
targets: ["承", "認", "承認"]
```

この場合、漢字としては `承` と `認` に触れたことになる。

---

## 9. 注意点

### 9.1 シリーズは固定しすぎない

50セットの枠は、最初から完全固定しない。
運用しながら、以下は調整してよい。

```text
シリーズごとのセット数
対象漢字
問題数
Set 045以降の総合演習内容
```

### 9.2 ただし番号帯は維持する

大きな方向性として、以下の番号帯は維持する。

```text
001〜010：基本熟語・日常語
011〜017：学校生活・学習語彙
018〜025：社会・制度・抽象語
026〜031：自然・理科・数量語彙
032〜038：同音異字・使い分け
039〜044：送り仮名・訓読み
045〜050：漢字知識・総合演習
```

番号帯を維持することで、教材全体の見通しがよくなる。
