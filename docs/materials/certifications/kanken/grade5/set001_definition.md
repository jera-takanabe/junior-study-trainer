# 漢検5級 Set 001 定義 v0.2

## 1. 目的

この文書は、漢検5級問題データの正式な教材セットとして、`Set 001：基本熟語・日常語 01` の対象範囲・作成方針・問題構成を定義する。

前回作成した `questions_kanken5_set01_v0_3_30` は、アプリ動作確認を兼ねた draft セットである。
今後は、本定義に基づいて、正式な `Set 001` として整備していく。

---

## 2. 前提

漢検5級は、小学校6年生修了程度を対象とし、対象漢字数は1026字である。
主な出題内容には、漢字の読み、書取、部首・部首名、筆順・画数、送り仮名、対義語・類義語、同音・同訓異字、誤字訂正、四字熟語、熟語の構成などが含まれる。

このため、単に「漢字1字を覚える」だけでなく、以下を扱う必要がある。

```text
読み
書き
語句の意味
文脈での使い分け
同音異字
送り仮名
熟語
四字熟語
```

## 4. Set の考え方

### 4.1 Set とは

`Set` は、漢字グループ単位の教材パックである。

```text
Set = 対象漢字20字前後 + 関連する読み・書き・意味・使い分け問題
```

### 4.2 Set の目的

1つの Set は、以下を目的とする。

```text
・対象漢字に一度は触れる
・読みと書きを確認する
・熟語や文脈の中で使えるようにする
・苦手問題を記録し、再出題できるようにする
```

### 4.3 Set 数の目安

1026字を20字前後ずつ扱う場合、最終的には50セット前後になる。

```text
1026字 ÷ 20字 ≒ 52セット
```

```text

重・険・従・温・結・果・責・任・供・給
```

---

## 7. 対象漢字の選定理由

### 7.1 熟語を作りやすい

以下のように、日常語・学校語彙として使いやすい熟語を作れる。

```text
承認
誕生
異常
勤勉
簡単
貴重
険しい
従う
温める
結果
責任
供給
```

### 7.2 読み・書き両方に向く

読み問題にも書き取り問題にも展開しやすい。

例：

```text
読み：『承認』の読みは？
書き：計画をショウニンする。
意味：『貴重』の意味は？
送り仮名：けわしい → 険しい
```

### 7.3 移動用と机上練習用に分けやすい

スピンバイク・車移動中は読みや意味選択、机上では書き取りや送り仮名にできる。


## 11. ID 命名ルール

Set 001 では、50セット規模での運用を前提に、set番号を含む正式IDを使う。

基本形：

```text
kanken5-s001-{type}-{連番3桁}
```

例：

```text
kanken5-s001-read-001
kanken5-s001-write-001
kanken5-s001-meaning-001
kanken5-s001-antonym-001
kanken5-s001-synonym-001
kanken5-s001-homophone-001
kanken5-s001-okuri-001
kanken5-s001-error-001
```

旧draft形式：

```text
kanken5-read-001
kanken5-write-001
```

は、試験運用用として扱い、正式採用データでは使わない。

理由：

```text
・Set単位で管理しやすい
・将来ファイル分割しやすい
・ID重複を避けやすい
・どのセット由来の問題か分かる
・進捗データと問題データの対応が追いやすい
```

## 12. Set 001 の採用判定

Set 001 を正式採用する前に、以下を確認する。

```text
1. v0.3アプリで読み込める
2. カテゴリ・出題タイプが正しく表示される
3. 進捗サマリーが正しく更新される
4. 苦手だけ出題できる
5. 進捗CSVに domain / exam / questionType / targets が出る
6. targets に対象漢字20字がすべて含まれる
7. 問題文が曖昧でない
8. 移動用と書き練習用の区別が妥当
9. ID形式が kanken5-s001-{type}-{連番3桁} になっている
```

## 13. Set 001 の targets 方針

Set 001 では、以下の両方を targets に含める。

```text
漢字1字
熟語・語句
```

例：

```javascript
targets: ["承", "認", "承認"]
```

これにより、以下の両方を将来的に集計できる。

```text
・1026字中、どの漢字に触れたか
・どの熟語・語句が苦手か
```

## 14. Set 001 のカテゴリ・level 方針


### 14.2 level

使う level は以下を基本とする。

```text

選択・類義語

送り仮名
```

四字熟語、部首、画数、熟語の構成は、Set 001 では必須にしない。
必要に応じて後続セットまたは総合演習セットで扱う。
