# Project Overview

## 1. プロジェクト名

Junior Study Trainer

## 2. 目的

Junior Study Trainer は、中学生向けのタイマー付き一問一答・反復学習アプリである。

移動中、スピンバイク中、家庭学習中など、書くことが難しい場面でも学習できるようにする。

また、机に向かう学習では、書き取り問題や復習問題にも使えるようにする。

## 3. 対象範囲

本プロジェクトは、以下の教材カテゴリを扱えることを目指す。

- 学校教科
  - 国語
  - 数学
  - 理科
  - 社会
  - 英語
- 外部検定
  - 漢検
  - 英検
  - 数検
- 拡張教材
  - 苦手克服
  - 定期テスト対策
  - 親子学習
  - スポーツ・生活習慣関連

## 4. 現在の実装

現在は、プロトタイプ教材として漢検5級を実装している。

- 漢検5級
- Set 001〜010
- 合計438問予定

## 5. 設計方針

アプリ本体は、漢検専用にしない。

問題データを差し替えることで、学校教科・検定・拡張教材を扱えるようにする。

## 6. 現時点の運用方針

当面は、ローカルHTML + JavaScriptで運用する。

- app/index.html
- app/questions.js

問題データは、当面は1本の questions.js として扱う。

ただし、Git上では将来の分割管理を意識して、教材カテゴリ・セット単位でも整理できる構成にする。
