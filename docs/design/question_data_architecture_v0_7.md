# Question Data Architecture v0.7

## 1. この設計書の目的

この設計書は、Junior Study Trainer における問題データ構造と配置方針を、v0.7 時点の前提として整理するためのものである。

現在のアプリは、漢検5級 Set001〜Set010 のようなテキスト中心の一問一答には対応できている。一方で、今後追加したい学校教科の問題では、図、表、グラフ、数式、長文、会話文、地図、年表などを扱う可能性がある。

そのため、学校教科の問題作成に進む前に、現行の `app/questions.js` 1本運用をどのように位置づけるか、問題データをどの単位で分割するか、実行時にどこから読み込むか、将来的に `data/` 配下の教材データと `app/` 配下の実行用データをどう関係づけるかを整理する。

v0.7 では、すぐに大規模な実装変更を行うのではなく、次の学校教科プロトタイプに進むための暫定方針を決めることを目的とする。
