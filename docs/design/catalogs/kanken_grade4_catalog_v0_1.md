# 漢検4級 学習内容カタログ v0.1

## 1. この文書の目的

この文書は、Junior Study Trainer における「漢検4級」対策の学習内容を整理するためのカタログである。

漢検5級は受験済みで、合否発表待ちである。合否にかかわらず、2026年10月中旬ごろに漢検4級を受験する前提で作成する。

漢検は学校教科とは別に、漢字だけを先取りして学習する検定として扱う。

目的は、問題データをすぐ作ることではなく、次の作業の前提をそろえることである。

- 漢検4級の公式出題範囲を正本として整理する
- 5級から4級で増える内容を明確にする
- 読み、書き、部首、送り仮名、対義語・類義語、同音・同訓異字、誤字訂正、四字熟語、熟語の構成を分けて扱う
- 学校教科の国語とは独立して、先取り学習できるようにする
- 紙で印刷する問題一覧・解答一覧への出力を妨げない構造にする
- 漢検3級へ進むときに拡張しやすい構造にする

このカタログは v0.1 であり、実装用データ形式ではない。

## 4. 5級との関係

漢検5級は、小学校6年修了程度の漢字を対象とする。4級では、中学校在学程度の漢字が加わる。

```text
漢検5級：
  小学校6年修了程度
  小学校配当漢字の総確認
  合否発表待ち

漢検4級：
  10月中旬ごろ受験予定
  中学校在学程度
  学校教科とは別に先取り
  小学校漢字を土台に、中学漢字・熟語・語彙力を広げる
```

5級の合否にかかわらず、4級学習へ進む。ただし、5級範囲の読み書きミスは、4級学習中に戻って補強する。

## 16. 熟語の構成

### 16.1 学習テーマ

```text
熟語の構成
├─ 似た意味の漢字を重ねる
├─ 反対の意味の漢字を重ねる
├─ 上の字が下の字を修飾する
├─ 下の字が上の字の目的語になる
├─ 主語・述語の関係
├─ 打ち消しの意味を含む熟語
├─ 接頭語・接尾語的な漢字
└─ 熟語の意味から構成を考える
```

### 16.2 タグ例

```text
domainTags:
  熟語の構成

questionTypeTags:
  分類、選択、説明

knowledgeTags:
  熟語、構成、修飾、反対、類似、主述、目的語

skillTags:
  熟語を分析する、意味から構成を考える、分類する
```

### 16.3 注意点

熟語の構成は、丸暗記よりも「どういう関係で二字が並んでいるか」を考えさせる。
