# 数検（算数検定）6級 学習内容カタログ v0.1

## 1. この文書の目的

この文書は、Junior Study Trainer における「数検（算数検定）6級」対策の学習内容を整理するためのカタログである。

目的は、問題データをすぐ作ることではなく、次の作業の前提をそろえることである。

- 数検6級の出題範囲を、公式情報を正本として整理する
- 小学校算数の抜け漏れ確認に使えるようにする
- 問題セット、出典、範囲、進捗管理を分離できるようにする
- 数検5級へ進むときに拡張しやすい構造にする
- 印刷用問題一覧・解答一覧への出力を妨げない構造にする

このカタログは v0.1 であり、実装用データ形式ではない。

## 3. 作成単位と拡張方針

このカタログは「数検 × 6級」を単位として作成する。

ただし、数検5級では中学1年程度の内容が追加されるため、次の考え方を持たせる。

```text
prerequisiteFromLowerGrades:
  小学校5年程度までの算数内容

newAtThisGrade:
  小学校6年程度の算数内容

emphasizedAtThisGrade:
  比、比例・反比例、円、立体、資料、単位、縮図・拡大図、分数を含む計算

nextGradePreparation:
  正負の数、文字式、一次方程式、作図、空間図形、比例・反比例、度数分布
```

## 4. 数検6級カタログの基本軸

数検6級では、学校教科の数学カタログと近いが、検定として次の軸を持たせる。

| 軸 | 役割 |
|---|---|
| domainTags | 数と計算、図形、測定、変化と関係、データの活用、文章題など |
| questionTypeTags | 計算、選択、穴埋め、作図的判断、文章題、資料読解など |
| knowledgeTags | 分数、比、円の面積、体積、比例、反比例、資料整理など |
| skillTags | 計算する、比較する、読み取る、式にする、図形を理解する、生活場面で使うなど |
| nextGradeTags | 5級・中1数学へつながる内容 |

## 18. 既存問題との関係

既存の学校数学カタログや一問一答問題は、このカタログの作成元ではなく、後から対応付ける対象とする。

方針：

- 公式範囲を基準に数検6級カタログを作る
- 既存問題は後で domainTags / questionTypeTags / knowledgeTags / skillTags に紐づける
- 不足テーマはカタログとの対応で発見する
- 6級対策では、計算だけでなく文章題・図形・表グラフも扱う
