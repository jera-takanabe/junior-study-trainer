# 中1数学 学習内容カタログ v0.1

## 1. 位置づけ

この文書は、中学1年数学について、現在使用している教科書全体を基準に作成した学習内容カタログの初版である。

既存問題データから逆算して作成した分類ではなく、教科書全体を先に確認し、学習内容として扱いやすい構造へ再整理したものである。

このカタログは、今後の問題作成、既存問題の移行、定期テスト範囲抽出、学習者UI設計の基準として使用する。

## 2. 基本方針

中1数学では、教科書の章立てを尊重しつつ、学習内容としては次の4領域に整理する。

- 数と式
- 関数
- 図形
- データの活用

また、数学では理科と異なり、共通技能を大きな独立単元として扱いすぎない。

主テーマは各単元内に置き、次のような横断技能は `skills` や副テーマとして問題に付与する。

- 式で表す
- 表で整理する
- グラフで表す
- 図で整理する
- 図を読み取る
- 作図する
- 立体を平面に直して考える
- 条件を満たすものを探す
- 解や答えが条件に合うか確かめる
- 理由を説明する
- 根拠をもって判断する

## 18. 教科書ページ範囲との関係

学習内容分類と教科書上のページ位置は分けて管理する。

定期テスト範囲がページで指定された場合は、分類ではなく `textbookAnchors` によって対象問題を抽出する。

例：

```text
問題
├─ 主テーマ：数と式 ＞ 方程式 ＞ 速さ・時間・道のりの問題
├─ skills：数量関係を見つける、方程式を立てる、解が条件に合うか確認する
└─ textbookAnchors：数学教科書 p.○○〜p.○○
```

これにより、分類上は文章題や活用問題として扱う問題でも、該当ページがテスト範囲に含まれていれば抽出対象にできる。

## 19. 現行問題との関係

現行の数学問題は、このカタログを確定した後で対応付ける。

既存問題に合わせてカタログを変えることはしない。

既存問題は、確定済みカタログへ後から差し込む。

想定される対応は次のとおり。

```text
第1回定期テスト用の正負の数・文字の式の問題
→ 数と式 ＞ 正の数・負の数
→ 数と式 ＞ 文字の式
```

文章題や説明問題は、単元の末尾にまとめず、該当する利用・説明テーマへ対応付ける。

## 20. 理科カタログとの比較

理科では、次の分離が重要であった。

```text
知識
観察
実験操作
資料活用
```

数学では、次の分離が重要である。

```text
概念
操作・計算技能
表現・説明
利用・文章題
```

また、理科では共通技能を独立領域として扱う比重が高い。

一方、数学では主テーマを各単元内に置き、共通技能は `skills` や副テーマとして付与する方が使いやすい。
