# 数検5級 学習内容カタログ v0.1

## 1. この文書の目的

この文書は、Junior Study Trainer における「数検5級」対策の学習内容を整理するためのカタログである。

数検5級は、中学1年範囲を先取りして受験する本命級として扱う。数検6級は、小学校算数の習熟度確認・抜け漏れ確認として扱い、6級受験前後にかかわらず、5級学習へ進める前提とする。

このカタログの目的は次の通り。

- 数検5級の公式出題範囲を正本として整理する
- 中1数学の先取り学習計画と接続する
- 1次：計算技能検定、2次：数理技能検定を分けて扱えるようにする
- 数検6級からの継承内容と、5級で新しく増える内容を明確にする
- 数検4級へ進むときに拡張しやすい構造にする
- 紙で印刷する問題一覧・解答一覧への出力を妨げない構造にする

このカタログは v0.1 であり、実装用データ形式ではない。

## 12. 1次・2次の学習方針

### 12.1 1次：計算技能

1次は、正確性とスピードを重視する。

```text
1次で重視すること
├─ 符号ミスを減らす
├─ 計算順序を間違えない
├─ 文字式を正しく整理する
├─ 方程式を安定して解く
├─ 比例・反比例の式を作る
└─ 30問を時間内に処理する
```

### 12.2 2次：数理技能

2次は、条件整理・図表読解・説明力を重視する。

```text
2次で重視すること
├─ 文章から条件を拾う
├─ 何を文字で置くか決める
├─ 方程式を立てる
├─ 図形の性質を使う
├─ グラフや表を読む
├─ 作図の意味を理解する
└─ 考え方を紙に残す
```
