# 中1理科 学習内容カタログ v0.1

## 1. 位置づけ

この文書は、中学1年理科について、現在使用している教科書全体を基準に作成した学習内容カタログの初版である。

既存問題データから逆算して作成した分類ではなく、教科書全体を先に確認し、学習内容として扱いやすい構造へ再整理したものである。

このカタログは、今後の問題作成、既存問題の移行、定期テスト範囲抽出、学習者UI設計の基準として使用する。

## 2. 基本方針

中1理科では、教科書全体を次の2種類に分けて扱う。

- 本文単元
  - 生命
  - 地球
  - 物質
  - エネルギー

- 横断的な技能・資料
  - 探究
  - 観察
  - 実験操作
  - 資料活用
  - 表・グラフ
  - 計算
  - 発表

教科書の章立てやページ範囲は、定期テスト範囲の抽出と網羅性確認に使う。

一方、学習内容カタログでは、学習者が復習しやすい意味的まとまりへ再構成する。

章末の「学習のまとめ」「力だめし」「発展」「コラム」「サイエンス資料」は、原則として独立した学習テーマにはしない。関連テーマの問題作成、網羅性確認、応用・発展問題の参考資料として扱う。

## 3. 上位構造

```text
中学校
└─ 1年
   └─ 理科
      ├─ 理科の学び方・探究技能
      ├─ 自然観察の技能
      ├─ 理科の共通技能・実験操作
      ├─ 理科の共通技能・資料活用
      ├─ 生命
      │  └─ いろいろな生物とその共通点
      ├─ 地球
      │  └─ 活きている地球
      ├─ 物質
      │  └─ 身のまわりの物質
      └─ エネルギー
         └─ 光・音・力による現象
```

共通技能を各単元の中に埋め込まず、横断的な学習内容として扱う。

ただし、定期テスト範囲の抽出では、学習分類ではなく教科書ページとの対応を使用する。そのため、分類上は共通技能に属する問題であっても、該当ページがテスト範囲内であれば抽出対象にできる。


## 12. 教材上の要素の扱い

### 本文・観察・実験

学習テーマの主な根拠にする。

### 学習のまとめ

網羅性確認と復習問題作成の参考にする。

### 力だめし

応用・統合問題の参考にする。

### 発展・コラム

発展問題または関連資料として扱う。

### 学年末総合問題

年間総復習・模擬問題集の参考にする。

### 索引・QR案内・教科書の使い方

教材構造には登録しても、原則として学習テーマにはしない。

## 13. 教科書ページ範囲との関係

学習内容分類と教科書上のページ位置は分けて管理する。

定期テスト範囲がページで指定された場合は、分類ではなく `textbookAnchors` によって対象問題を抽出する。

例：

```text
問題
├─ 主テーマ：理科の共通技能・実験操作 ＞ ガスバーナーの使い方
└─ textbookAnchors：理科教科書 p.○○〜p.○○
```

これにより、分類上は共通技能に属する問題であっても、該当ページがテスト範囲に含まれていれば抽出対象にできる。

## 14. 現行問題との関係

現行の理科問題は、このカタログを確定した後で対応付ける。

```text
S001 観察の基本・観察器具
→ 理科の学び方・探究技能
→ 自然観察の技能
→ 理科の共通技能・実験操作

S002 植物の特徴
→ 生命 ＞ 植物の特徴と分類

S003 動物の分類
→ 生命 ＞ 動物の特徴と分類
```

ただし、既存問題に合わせてカタログを変えることはしない。

既存問題は、確定済みカタログへ後から差し込む。
