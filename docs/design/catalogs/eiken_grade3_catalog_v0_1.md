# 英検3級 学習内容カタログ v0.1

## 1. この文書の目的

この文書は、Junior Study Trainer における「英検3級」対策の学習内容を整理するためのカタログである。

英検4級は合格済みとし、次回は2026年9月末ごろの日程を想定して英検3級を受験する前提で作成する。

目的は、問題データをすぐ作ることではなく、次の作業の前提をそろえることである。

- 英検3級の公式出題形式を正本として整理する
- 4級から3級で増える内容を明確にする
- リーディング、ライティング、リスニング、スピーキングを分けて扱えるようにする
- 音声再生・音声入力・AI評価を将来機能として扱えるようにする
- 紙で印刷する問題一覧・解答一覧への出力を妨げない構造にする
- 準2級へ進むときに拡張しやすい構造にする

このカタログは v0.1 であり、実装用データ形式ではない。

## 4. 4級との関係

英検4級は合格済みである。3級では、4級までの内容を前提として、書く力・話す力が本格的に必要になる。

```text
英検4級：
  基本語彙
  基本文法
  会話表現
  読解
  リスニング

英検3級：
  中学卒業程度
  リーディング
  ライティング
  リスニング
  二次面接スピーキング
  Eメール返信
  意見英作文
  音読
  パッセージ質問
  イラスト描写
  自分の意見・経験を答える
```

```text

├─ 文脈に合う語彙を選ぶ
├─ 文脈に合う熟語を選ぶ
├─ 基本文法を使って選ぶ
├─ 品詞を判断する
├─ 時制を判断する
├─ 前後関係を読む
└─ 会話文脈を読む
```

```text

```

```text
会話文の空所補充
```

