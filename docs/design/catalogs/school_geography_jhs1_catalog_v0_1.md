# 中1地理 学習内容カタログ v0.1

## 1. 位置づけ

この文書は、中学1年地理について、現在使用している教科書全体を基準に作成した学習内容カタログの初版である。

既存問題データから逆算して作成した分類ではなく、教科書全体を先に確認し、学習内容として扱いやすい構造へ再整理したものである。

このカタログは、今後の問題作成、既存問題の移行、定期テスト範囲抽出、学習者UI設計の基準として使用する。

## 2. 基本方針

中1地理では、教科書の編・章構成を尊重しつつ、学習内容としては次の4領域に整理する。

- 地理の学び方・資料活用
- 世界と日本の地域構成
- 世界のさまざまな地域
- 日本のさまざまな地域

地理では、理科の「観察・実験」、数学の「解法・操作技能」とは異なり、次の3軸が重要である。

```text
地域軸：
世界、日本、州、地方、都道府県、身近な地域

内容軸：
位置、自然環境、人口、産業、文化、交通、災害、防災、地域課題、持続可能性

技能軸：
地図、写真、統計、グラフ、気候グラフ、地形図、比較、調査、説明、発表
```

主テーマは教科書の学習内容に置き、地域・内容・技能は `regionTags`、`contentTags`、`skills` として付与できるようにする。

例：

```text
主テーマ：
日本のさまざまな地域 ＞ 日本の諸地域 ＞ 関東地方 ＞ 東京大都市圏

regionTags：
日本、関東地方、東京大都市圏

contentTags：
人口、都市、交通、産業、地域課題

skills：
地図を読む、統計資料を読む、地域を比較する、課題を説明する
```

```text
├─ 資料を集める
├─ 資料を読み取る
├─ 要因を考える
├─ 解決策を考える
├─ 根拠をもって説明する
└─ 発表・振り返りを行う
```

これらは独立テーマとしても扱えるが、基本的には地域調査・持続可能な地域の在り方の中で扱う。

## 17. 巻末資料の扱い

巻末資料は、本文単元の学習テーマには直接混ぜない。

次のように、出典・補助資料・資料活用問題の材料として扱う。

```text
巻末資料
├─ 用語解説
├─ 地名・事項索引
├─ 統計資料
├─ 世界の主な家畜・農産物・鉱産資源
├─ デジタルコンテンツの活用案内
└─ 奥付・裏表紙
```

### 用語解説

用語確認・一問一答の補助資料として扱う。

### 地名・事項索引

教材構造上の参照資料として扱う。原則として学習テーマにはしない。

### 統計資料

資料読解・比較問題の出典として扱う。

例：

```text
主テーマ：
日本のさまざまな地域 ＞ 日本の地域的特色 ＞ 人口から見た日本

skills：
統計資料を読む、分布を比較する

sources：
地理教科書 巻末 都道府県統計
```

### 世界の主な家畜・農産物・鉱産資源

世界の諸地域、産業、貿易、資源問題の参照資料として扱う。

### デジタルコンテンツ

出典・補助教材として記録する。原則として学習テーマにはしない。

## 18. 教科書ページ範囲との関係

学習内容分類と教科書上のページ位置は分けて管理する。

定期テスト範囲がページで指定された場合は、分類ではなく `textbookAnchors` によって対象問題を抽出する。

例：

```text
問題
├─ 主テーマ：世界と日本の地域構成 ＞ 日本の姿 ＞ 時差から見た日本の位置 ＞ 時差の計算
├─ skills：地図を読む、経度差から考える、計算する
├─ regionTags：日本、世界
├─ contentTags：位置、時差、経度
└─ textbookAnchors：地理教科書 p.○○〜p.○○
```

これにより、分類上は資料活用や地理的技能として扱う問題でも、該当ページがテスト範囲に含まれていれば抽出対象にできる。

## 19. 現行問題との関係

現行の地理問題は、このカタログを確定した後で対応付ける。

既存問題に合わせてカタログを変えることはしない。

既存問題は、確定済みカタログへ後から差し込む。

想定される対応は次のとおり。

```text
世界の姿に関する既存問題
→ 世界と日本の地域構成 ＞ 世界の姿

日本の姿に関する既存問題
→ 世界と日本の地域構成 ＞ 日本の姿

人々の生活と環境に関する既存問題
→ 世界のさまざまな地域 ＞ 人々の生活と環境
```

## 20. 理科・数学カタログとの比較

理科では、次の分離が重要であった。

```text
知識
観察
実験操作
資料活用
```

数学では、次の分離が重要であった。

```text
概念
操作・計算技能
表現・説明
利用・文章題
```

地理では、次の分離が重要である。

```text
地域
資料読解
比較
因果関係
地域課題
地理的探究
```

地理では特に、地域軸・内容軸・技能軸を分けて持つことが重要である。
