# 中1歴史 学習内容カタログ v0.1

## 1. この文書の目的

この文書は、中学1年の歴史教科書をもとに、Junior Study Trainer で扱う学習内容を整理するためのカタログである。

目的は、問題データをすぐ作ることではなく、次の作業の前提をそろえることである。

- 学習テーマを、教科書ページや既存問題から独立して整理する
- 問題セット、出典、ページ範囲、進捗管理を分離できるようにする
- 歴史特有の「時代軸」「因果関係」「資料活用」を扱えるようにする
- 後で目的別問題集を作るときに、抽出条件や明示ID一覧を設計しやすくする

このカタログは v0.1 であり、実装用データ形式ではない。

## 3. 歴史カタログの基本設計

歴史は、地理以上に時代軸が重要である。

### 3.1 主軸

歴史の主軸は次の4つとする。

| 軸 | 役割 |
|---|---|
| 時代軸 | いつの出来事か、どの時代の特色かを整理する |
| 地域軸 | 日本、東アジア、世界、身近な地域との関係を整理する |
| 内容軸 | 政治、社会、経済、文化、外交、宗教、制度、戦争などを整理する |
| 技能軸 | 年表、地図、史料、写真・絵画資料、比較、因果関係、説明、探究を整理する |

### 3.2 問題データで想定する属性

将来の問題データでは、少なくとも次の属性を持てるようにする。

```text
primaryTheme:
  例）中世の日本 ＞ 武士の政権の成立 ＞ 鎌倉幕府の成立

eraTags:
  例）鎌倉時代

regionTags:
  例）日本、鎌倉

contentTags:
  例）政治、武士、幕府、土地支配

skills:
  例）年表を読む、制度を説明する、因果関係を説明する
```

### 3.3 歴史で重視する問い

歴史では、用語暗記だけでなく、次のような問いを扱えるようにする。

- なぜその出来事が起きたのか
- その出来事によって何が変わったのか
- その時代の特色は何か
- 日本と世界はどのように関係していたか
- 政治・社会・経済・文化はどのように結び付いていたか
- 現在の社会とどのようにつながっているか
